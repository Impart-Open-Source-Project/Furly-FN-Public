# Furly

一个基于 Flutter 开发的项目。  
该项目并未完成，您可以加入我们的开发。

## 开始

安装 flutter sdk(推荐使用 sidekick 进行 fvm 管理)。  
克隆本仓库，在根目录使用`flutter pub get`安装所需依赖。  
打开您觉得顺眼的编辑器，开始开发。

## 一些有用的东西  

使用以下命令生成新增的多语言配置
```flutter gen-l10n```

## amplify command

check status:

amplify status

sync cloud:

amplify push

code gen:

amplify codegen models
